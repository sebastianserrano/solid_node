> A server which displays the core design principles of software development (S.O.L.I.D)

## Description

Solid Node has the following routes:
<br/>
- Create worker
- Delete workers
- Create work order
- Assign work order
- Fetch work orders by:
  - Specific worker
  - Sorted by deadline
